# snie2012.github.io

This is a Jekyll site (Minima theme) for posts and pages.

## Local development

Run:

```
bundle install
bundle exec jekyll serve
```

Open `http://localhost:4000`.

## Structure
- `_posts/` Markdown posts
- `_layouts/` and `_includes/` templates
- `_sass/` theme and custom SCSS
- `assets/` CSS and images

## Plugins
- `jekyll-feed` for RSS/Atom
- `jekyll-sitemap` for sitemap.xml

## Configuration
See `_config.yml`. Set `url` to your site URL and update `title`/`description`.

## Deploy
Push to `main`; served by GitHub Pages.
